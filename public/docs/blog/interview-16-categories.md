## 建立你的前端知识图谱

在准备面试时，盲目刷题不如建立体系。我们将前端面试题梳理为 16 个核心分类：

### 1. JavaScript 基础
*   原型与原型链
*   作用域与闭包
*   `this` 指向与 `call/apply/bind`
*   ES6+ 新特性 (Promise, async/await, Proxy)

### 2. CSS 与 布局
*   盒模型 (Box Model)
*   Flexbox 与 Grid 布局
*   BFC (块级格式化上下文)
*   移动端适配 (rem, vw, media queries)

### 3. 框架核心 (Vue/React)
*   **Vue**: 响应式原理, diff 算法, computed vs watch
*   **React**: Fiber 架构, Hooks 原理, 合成事件

### 4. 浏览器原理
*   渲染流程 (DOM -> CSSOM -> Layout -> Paint)
*   事件循环 (Event Loop)
*   缓存策略 (强缓存 vs 协商缓存)

### 5. 工程化
*   Webpack / Vite 构建流程与优化
*   Babel 编译原理
*   模块化规范 (CommonJS vs ESM)

*(此处省略其他 11 个分类详情，包括网络、安全、性能优化、设计模式、算法、手写代码、项目经验、软技能等)*

### 总结
面试不仅是考核知识点，更是考察你的知识广度与深度。建议针对每个分类整理出自己的思维导图。
